# Screenshots

Drop your own screenshots here, then reference them in the main README's Screenshots section.

**Suggested shots, in priority order:**
1. `pipeline.png` — Pipeline tab mid-run, stage badges lit up (this is the single best screenshot for showing what the project actually does at a glance)
2. `home.png` — the homepage
3. `dashboard.png` — Dashboard with a few real runs in it
4. `dark-mode.png` — any tab in dark theme, to show the theme system works

**How to capture a clean one:**
- Use your browser's device toolbar (F12 → toggle device toolbar) set to a fixed width like 1440px, so the screenshot isn't cropped oddly
- Crop out the browser chrome (address bar, bookmarks) — just the app itself
- PNG, not JPEG — the UI has flat colors and text that compress much more cleanly as PNG
